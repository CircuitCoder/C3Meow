# C3meow

> C3Blog 的一个前端实现

## 构建
传统三板斧:

```bash
git clone https://github.com/CircuitCoder/C3Meow.git && cd C3Meow
npm install

# 编辑配置文件
cp src/config.example.js src/config.js
editor src/config.js

npm run build
```

然后你就可以在 `/dist` 目录下看到生成的文件了

## 开发
一言不合
```
npm run dev
```
